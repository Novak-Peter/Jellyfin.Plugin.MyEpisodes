# Project Plan: Jellyfin.Plugin.MyEpisodes

## Overview
This repository contains a Jellyfin plugin that synchronises watch status with **MyEpisodes.com**. The plugin allows per‑user credentials, tracks when a user marks an episode as watched in Jellyfin, and updates the corresponding entry on MyEpisodes via HTTP scraping.
